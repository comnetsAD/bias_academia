# Code Repository for "Causal evidence of racial and institutional biases in accessing paywalled articles and scientific data"

All code and data needed to replicate findings from studies 3 and 5 can be found under `/audit_experiments/`

All code needed to replicate findings from studies 2 and 4 can be found under `/observational_analyses/`. Data must be downloaded from [Figshare](https://figshare.com/articles/dataset/Causal_evidence_of_racial_and_institutional_biases_in_accessing_paywalled_articles_and_scientific_data/30126952?file=57963016).
